# **2. One-Page Investment Memo**

## **AdaRSS Investment Memo**

AdaRSS is building a Career Stability Intelligence Layer that classifies and predicts the long-term durability of skills in a rapidly evolving global economy. The platform addresses a structural inefficiency in labor markets: the absence of a reliable system to distinguish between durable, emergent, and obsolete skills.

### **Problem**

Workforce systems are reactive and fragmented. Education providers and governments lack predictive intelligence on which skills will remain relevant over time. As a result, individuals invest in skills that quickly lose economic value, while employers face increasing uncertainty in talent quality and readiness.

### **Solution**

AdaRSS introduces a machine learning–powered classification system that evaluates skills based on long-term economic stability. Skills are categorized into:

* Enduring Foundations
* Emergent Necessities
* Transient Skills

On top of this, AdaRSS provides a personalization engine that maps individual profiles to career trajectories and recommends strategic skill decisions.

### **Product**

We have built an early production system including:

* DistilBERT-based skill classification model
* Sentence-transformer similarity engine for career guidance
* FastAPI backend with authentication, API key management, and usage tracking
* React-based admin and client dashboards
* B2B subscription and API infrastructure

### **Wedge**

Initial adoption begins with technical professionals and high-frequency learners who require continuous skill decision-making. AdaRSS is delivered as an API-first platform for integration into existing workflows.

### **Expansion**

The system expands into:

* EdTech platforms for curriculum design
* Workforce agencies for employment planning
* Government systems for labor forecasting
* Enterprise HR systems for talent intelligence

### **Moat**

AdaRSS combines three layers:

1. Skill classification model trained on labor intelligence signals
2. Personalization engine based on semantic similarity
3. Infrastructure layer enabling enterprise integration

This combination creates a defensible workforce intelligence system, not a standalone tool.

### **Vision**

To become the global infrastructure layer for workforce intelligence and career stability prediction.
